# FreeBlock

An open source desktop website blocker

## Usage

- Show status of blocking lists: `freeblock status`
- Create new list: `freeblock list add [name]`
- Edit a list: `freeblock list edit [name]`
- Rename a list: `freeblock list rename [old] [new]`
- Remove a list: `freeblock list remove [name]`
- Enable a list: `freeblock block [name]`
- Disable a list: `freeblock unblock [name]`
